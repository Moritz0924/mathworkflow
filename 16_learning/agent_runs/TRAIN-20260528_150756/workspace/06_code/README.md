# 代码说明

## 运行方式

```bash
pip install -r requirements.txt
python run_all.py
```

## 目录说明

- `src/`: 主模型、数据处理、绘图函数。
- `notebooks/`: 探索性分析笔记本，可选。
- `execution_log.md`: 每次运行记录。

## 结果输出约定

- 每问结果输出到 `07_results/q*_results.csv`。
- 总指标输出到 `07_results/metrics_summary.csv`。
- 结果来源映射输出到 `07_results/result_source_map.csv`。
