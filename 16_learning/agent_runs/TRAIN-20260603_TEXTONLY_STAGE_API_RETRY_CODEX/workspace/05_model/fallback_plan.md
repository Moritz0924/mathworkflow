# 备选方案计划 (Fallback Plan)

本文档描述当主模型因数据、假设或计算原因失效时，应如何降级到备选模型或替代分析方案。

## Q1：数据预处理与特征工程
- **主模型失效场景**: 皮尔逊相关性未能识别任何显著特征（|r| < 0.2）。
  - **降级行动**: 改用 LASSO 回归进行特征选择，并检查多重共线性；若 LASSO 也无法收敛，则基于领域知识手动选择温度、湿度、风速作为候选特征。
- **数据缺失处理失效**: 若某些变量缺失率突然增加（如 > 20%），则直接丢弃该变量，仅使用完整度高的变量。

## Q2：PM2.5 浓度预测
- **主模型失效场景**: SARIMA 残差未通过白噪声检验（Ljung-Box p < 0.05），或测试集 MAPE > 30%。
  - **降级行动**: 尝试 SARIMAX 加入关键外生变量（温度、湿度）以捕捉额外结构；若仍失败，切换至 XGBoost 模型，并进行超参数网格搜索。
- **备选模型失效**: XGBoost 在测试集上仍然 MAPE > 30%。
  - **最终降级**: 采用朴素预测，即未来 24/48/72 小时的 PM2.5 预测值等于过去 24 小时的平均值；此基线主要用于对比，不作为最终推荐，但可确保预测方法不会比基准更差。
- **计算资源不足**: 若时间紧迫或算力有限，放弃 SARIMA 网格搜索，改用 auto_arima 自动定价，并限制最大差分阶数 d ≤ 2。

## Q3：减排策略优化
- **主模型失效场景**: 线性规划返回不可行解（约束冲突）。
  - **降级行动**: 逐步放宽目标浓度约束（B 值），或改变部分排放源为不可削减；若仍无解，采用目标规划（Goal Programming）允许超过目标浓度但施加惩罚。
- **成本参数不可靠**: 若减排成本系数 c_i 取自宽泛文献范围，且导致最优解对参数极其敏感。
  - **降级行动**: 采用鲁棒优化，将成本系数视为区间数，求解 min-max 后悔模型。
- **多目标需求明确**: 若决策者要求同时最小化成本和最大化就业或最小化其他污染物。
  - **降级行动**: 启用 NSGA-II 生成帕累托前沿，并由决策者根据偏好选择，但需注意此降级将增加图表表达和解释复杂度。

## 全局计划
- 所有降级方案均需记录在实验日志中，并在最终论文中说明选择主模型的理由以及因何原因启用了备选方案。
- 在任何阶段，如果发现原问题数据条件完全无法支撑任何模型，将停止建模该子问题，并在论文中作为局限性章节说明。
