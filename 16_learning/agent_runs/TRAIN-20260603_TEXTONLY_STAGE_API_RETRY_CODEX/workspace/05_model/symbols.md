# 符号说明 (Symbols Glossary)

本文档集中定义论文中将使用的核心符号，以确保全文一致。子问题或模型特有的符号可在对应章节补充说明。

| 符号 | 含义 | 单位 / 备注 |
|---|---|---|
| y_t | t 时刻的 PM2.5 浓度 | μg/m³ |
| T_t | t 时刻的温度 | °C |
| H_t | t 时刻的相对湿度 | % |
| W_t | t 时刻的风速 | m/s |
| P_t | t 时刻的气压 | hPa |
| s_t | 季节/时间特征（如小时、星期） | 类别/数值 |
| \hat{y}_t | t 时刻的预测 PM2.5 浓度 | μg/m³ |
| ε_t | 模型残差 | 同 y_t |
| d | 差分阶数 | 整数 |
| D | 季节性差分阶数 | 整数 |
| p, P | 自回归、季节性自回归阶数 | 整数 |
| q, Q | 移动平均、季节性移动平均阶数 | 整数 |
| x_i | 第 i 种排放源的削减比例 | % |
| c_i | 第 i 种排放源的单位削减成本系数 | 万元/百分比 |
| A_i | 第 i 种源对 PM2.5 的线性贡献系数 | (μg/m³)/百分比 |
| B | 目标 PM2.5 浓度上限 | μg/m³ |
| C_total | 总减排成本 | 万元 |
| RMSE | 均方根误差 | μg/m³ |
| MAE | 平均绝对误差 | μg/m³ |
| MAPE | 平均绝对百分比误差 | % |

**注**：具体模型中的参数（如 ARIMA 的 φ, θ, Φ, Θ）将在相应的公式和模型描述中明确定义。
