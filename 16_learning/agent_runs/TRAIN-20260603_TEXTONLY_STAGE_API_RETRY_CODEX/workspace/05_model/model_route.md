# 模型路线路由 (Model Route)

## 1. 总体策略
基于任务分解和数据质量报告，我们将问题划分为三个子问题，并针对每个子问题选择最适合的模型族。选择遵循以下原则：
- 数据可行性优先：模型假设必须与 EDA 发现一致。
- 可解释性：竞赛评审注重模型清晰度，避免过度复杂的黑箱。
- 复现性：模型需要明确的参数和随机种子，确保他人可复现。
- 评分风险：考虑常见评分点，如模型验证、敏感性分析和图表表达。

## 2. 子问题模型路由

### 子问题 1：数据预处理与特征工程
- **任务 ID**: Q1
- **描述**: 识别异常值、插补缺失值、选择与 PM2.5 浓度相关的关键特征。
- **数据字段**: PM2.5 浓度, 温度, 湿度, 风速, 气压, 季节标识
- **EDA 发现**: PM2.5 与温度呈负相关，与湿度正相关，存在明显的季节性和日周期；缺失率不超过 5%。
- **主模型**: 滑动窗口平滑（异常值处理）+ 皮尔逊相关性矩阵（特征选择）
  - 理由：简单直观，易于解释，计算成本低。
  - 验证：通过箱线图识别异常值，缺失值插补后通过交叉验证对比插补前后的分布。
- **备选模型**: LASSO 回归特征选择
  - 理由：若线性相关不显著或特征间存在多重共线性，LASSO 可自动筛选特征并控制过拟合。
  - 触发条件：皮尔逊相关系数大多低于 0.3 或 VIF 较高。
- **图表表达建议**: 热力图展示相关性，时间序列折线图展示原始与清洗后数据对比。

### 子问题 2：PM2.5 浓度预测
- **任务 ID**: Q2
- **描述**: 基于历史数据和气象条件，预测未来 24/48/72 小时的 PM2.5 浓度。
- **数据字段**: 历史 PM2.5 序列，温度，湿度，风速，时间特征（小时、星期、月份）
- **EDA 发现**: 序列平稳性检验显示一阶差分后平稳；ACF/PACF 显示明显的 24 小时季节性和缓慢衰减的自相关。
- **主模型**: SARIMA(p,d,q)(P,D,Q)_24
  - 理由：专长于季节性时间序列预测，参数有明确统计解释，复现性强。
  - 验证：使用滚动窗口时间序列交叉验证，评估 RMSE、MAE；通过 Ljung-Box 检验残差白噪声。
  - 公式对象：见公式合同 F1-ARIMA。
- **备选模型**: XGBoost 回归（特征工程丰富时采用机器学习）
  - 理由：若 SARIMA 误差过大或残差非白，XGBoost 能捕捉非线性关系，且可通过 SHAP 提供部分解释。
  - 触发条件：SARIMA 在测试集上 MAPE > 20% 或残差自相关显著。
- **降级方案**: 若 XGBoost 同样失败，使用朴素预测（过去 24 小时均值）作为基线。
- **图表表达建议**: 预测值与实际值对比折线图、残差分布直方图、误差指标随时间变化图。

### 子问题 3：减排策略优化
- **任务 ID**: Q3
- **描述**: 在预测基础上，确定各排放源的削减比例，以最小成本达到目标 PM2.5 浓度。
- **数据字段**: 排放源清单（工业、交通、扬尘等）、减排成本系数、各源对 PM2.5 的贡献系数（由预测模型或外部数据提供）
- **EDA 发现**: 假设各源贡献系数已知或通过敏感度分析推导；成本系数来自文献。
- **主模型**: 线性规划（LP）
  - 理由：问题具有明确的线性成本结构和约束，LP 可得全局最优解，易于解释和灵敏度分析。
  - 验证：通过改变约束值（目标浓度）、成本系数进行灵敏度分析，检查解的稳定性和合理性。
  - 公式对象：见公式合同 F2-LP。
- **备选模型**: 多目标优化（如 NSGA-II）
  - 理由：若需同时优化成本和另外的环境指标（如 SO2、NOx），LP 无法处理多个目标，可转入多目标进化算法。
  - 触发条件：决策者提出多个且相互冲突的目标。
- **降级方案**: 若 LP 无解（约束矛盾），采用目标规划放松硬约束，或转为手动情景分析。
- **图表表达建议**: 成本-效益曲线（不同削减比例下的成本与浓度关系），最优策略变量柱状图，灵敏度热力图。

## 3. 风险清单
1. **数据缺失风险**: 若关键气象变量缺失严重，可能影响预测精度，需依赖备选插补方案或舍弃该变量。
2. **平稳性假设风险**: SARIMA 要求差分后序列平稳，若实际序列存在结构性断点（如政策突变），可能导致预测失败。
3. **线性假设风险**: LP 假设成本与削减比例线性，且排放-浓度关系线性，实际中可能非线性，备选模型能缓解但增加复杂度。
4. **变量不可获取风险**: 排放源贡献系数若无法从数据中获得，需依赖假设或文献典型值，可能引入较大偏差，需在结论中注明。
5. **模型不可复现风险**: 深度学习备选方案若使用复杂网络且不固定种子，可复现性差；优先选用有固定随机状态的实现。
6. **符号混淆风险**: 已登记公式合同，但仍需注意不同文献中符号差异，论文中统一使用 symbols.md 定义。
