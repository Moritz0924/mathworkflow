# 润色规则 v1.0

> 本文件定义 polish 阶段的中文学术润色规则，所有润色操作必须遵守事实保持原则。

## 1. 受保护事实原子（不可更改）

- **数字**：所有数值、百分比、p 值、置信区间、样本量、指标值
- **单位**：所有物理单位、货币单位、统计单位
- **公式**：所有 LaTeX 公式、编号、变量定义
- **标签**：`\label{...}`、`\ref{...}`、`\cite{...}`
- **引用**：所有 `\cite{}` 键和文献条目
- **模型名**：所有模型名称、算法名称、软件包名称
- **结果含义**：所有结论的强度、方向、显著性水平
- **图表编号**：图/表序号及引用

## 2. 允许润色范围

- 句式和语序调整（被动→主动、长句拆分、冗余压缩）
- 段落衔接词优化（"此外"、"然而"、"因此"、"具体而言"）
- 中文学术表达规范化（消除口语化、统一术语、提升正式度）
- 消除歧义表达（但不改变原意）
- 修正错别字和标点（中文标点规范）

## 3. 禁止润色范围

- 不得增加新的事实陈述
- 不得删除已有的数据或结论
- 不得将弱结论强化为强结论
- 不得将中文润色为英文
- 不得修改 LaTeX 结构命令
- 不得修改 citation key
- 不得新增或删除引用

## 4. 弱证据句处理规则

对于含有限定性词的句子（可能、似乎、倾向于、在特定条件下、初步表明），润色时：
- 保留所有限定词
- 不得删除或替换为确定性表达
- 可优化句式但保留限定语义

## 5. 差异检查流程

每个润色片段完成后，必须执行以下检查：
1. `changed_numbers`: 逐数字比对
2. `changed_units`: 逐单位比对
3. `changed_formulas`: 逐公式比对
4. `changed_labels`: 逐 `\label`/`\ref` 比对
5. `changed_refs`: 逐 `\cite` 比对
6. `changed_citations`: 逐文献条目比对
7. `changed_model_names`: 逐模型名比对
8. `changed_result_meanings`: 结果含义逐句比对

若任一项为真，回滚该片段并记录到 `polish_diff_check.csv`。

## 6. 中文表达规范

- 使用全角中文标点：，。；：？！（）【】《》
- 英文术语首次出现加括号注释
- 数字与单位间加空格（如 25 °C）
- 图表引用格式：图~\ref{fig:xxx}、表~\ref{tab:xxx}
