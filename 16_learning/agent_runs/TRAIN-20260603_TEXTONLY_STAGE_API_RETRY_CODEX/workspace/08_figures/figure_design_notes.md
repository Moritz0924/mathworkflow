# 图表设计说明

## 设计原则
- 所有图表必须服务于已登记的结果或证据。
- 使用非默认配色（muted_blue_gold, teal_orange_diverging, deep_blue_vermilion）。
- 图中文字必须为中文（指标缩写、变量名除外）。
- 质量分目标 >= 4.2。

## 图表清单
1. **FIG-001 指标体系层级图**：展示三层结构，绑定 RES-001（综合得分）。用于第3节。
2. **FIG-002 权重热力图**：显示各准则下指标权重，绑定 RES-002（权重分配）。用于第4节。
3. **FIG-003 敏感性热力图**：参数灵敏度，绑定 RES-003（敏感性分析）。用于第5节。

## 技术方案
- 当前为 SVG 占位符；实际图形应由 matplotlib/seaborn 生成，采用相应模板。
- 中文支持需设置 SimHei 字体或系统默认中文字体。
