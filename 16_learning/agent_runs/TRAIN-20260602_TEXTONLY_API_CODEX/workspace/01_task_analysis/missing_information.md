# 缺失信息与风险评估

## 1. 数据层面缺失
- **成分标准化定义**：题目未明确各氧化物名称的统一标准（如 Fe₂O₃ 与 FeO 未区分），附件各列含义只能依靠常见考古学惯例，缺乏官方字典。
- **风化定义**：“风化”为二分变量，未说明是物理风化、化学风化还是综合判断，可能在不同文物上有不同判定标准。
- **颜色标注规范**：颜色名称不统一，如“蓝紫”“紫”“深紫”的界限模糊，是否可归并需进一步分析。
- **附件2 亚类量化规则**：多使用“较高”“较低”等主观词，需人工转化为数值区间或统计检验，转换标准不确定。

## 2. 题目要求歧义
- **“分析表面风化与玻璃类型、颜色、化学成分的关系”**：关系如何衡量？是关联性、预测性还是因果？需用统计检验还是回归/分类？未明确。
- **“对附件3预测风化前和风化后的玻璃类型”**：需要给出每个文物的两种预测？若风化前类型已知，则预测风化后；若不已知，则需推断风化前类型，但附件3没有风化标签，如何定义“风化前”依赖于对风化机理的建模，存在解释多样性。
- **亚类划分与附件2一致性**：若附件2描述产生多种合理划分（如不同阈值），哪种算符合“附件2的分类逻辑”？评分标准未知。

## 3. 交付与评分模糊点
- **支撑材料范围**：是否要求提供完整的模型推导、中间结果表？不明确。
- **论文格式**：字数、图表数量是否有竞赛惯例但未在题面明示？
- **运行环境**：未要求提供软件版本或包依赖，但通常裁判需复现，可能隐含要求。

## 4. 高风险项
| 风险项 | 影响 | 建议 |
|--------|------|------|
| 成分合计偏离 | 预处理方式影响后续分析（归一化、删减） | EDA 阶段需对待合计值进行分析，记录假设 |
| 风化定义主观 | 建模标签噪声大，影响分类精度 | 考虑使用半监督或对风化概率建模 |
| 附件3 无风化/颜色 | 预测风化前后类型的任务定义不清 | 需在建模前明确假设，并设计验证方法 |
| 亚类量化阈值 | 划分结果因人而异，影响一致性 | 建立基于数据的聚类与专家规则结合的方案 |
| 缺失值处理 | 微量成分缺失可能是真零或未检测 | 需要化学背景知识决策填补策略 |
