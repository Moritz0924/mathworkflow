# 赛题整理：2022 高教社杯全国大学生数学建模竞赛 C 题

## 1. 题目概括
古代玻璃制品的成分分析与鉴别。通过考古发现的古玻璃文物成分数据，研究不同种类古玻璃的特征、表面风化对其成分的影响，并对未知文物进行类型鉴别和亚类划分。

## 2. 研究对象
- **对象**：中国境内发现的古代玻璃制品（出土/馆藏）。
- **时空范围**：中国古代（具体年代未明，但文物涵盖战国至明代的玻璃器物）。
- **样本属性**：编号、是否风化（是/否）、颜色（多类）、类型（高钾玻璃、铅钡玻璃）。

## 3. 变量清单

### 3.1 成分变量（数值型，单位：%）
- SiO₂, Na₂O, K₂O, CaO, MgO, Al₂O₃, Fe₂O₃, PbO, BaO, P₂O₅, SnO₂, TiO₂, ZnO, CuO, MnO, SrO, Rb₂O, As₂O₃, SO₃, Cl, ZrO₂, …（共 20+ 种氧化物或元素含量）
- 部分样本存在缺失值（用空格或“-”表示）。

### 3.2 表面特征变量
- **风化标识**：二元变量（风化/未风化）。
- **颜色**：多类别（如浅蓝、蓝紫、黑、绿、青绿、蓝绿、黄等），部分可归并。

### 3.3 标签变量
- **类型**：高钾玻璃、铅钡玻璃（附件1 含标注）。
- **亚类**：附件2 给出了高钾和铅钡内部的亚类划分及特征描述。

## 4. 附件说明
详见 `00_problem/attachments_overview.md`。

## 5. 题目要求与交付物

### 问题 1
分析表面风化与玻璃类型、颜色、化学成分的关系（图表、统计量）。

### 问题 2
对附件3中未知文物进行类型鉴别（高钾/铅钡），提交结果表（文物编号＋类型）。

### 问题 3
利用附件2给出的亚类信息，对附件1中的文物进行亚类划分，分析各亚类的化学成分规律。

### 问题 4
对附件3中的文物预测表层风化前和风化后的玻璃类型，并分析预测结果的合理性。

**提交格式**：论文 PDF + 支撑材料（含代码、结果表等），符合竞赛规范。

## 6. 显式约束
- 附件1 为训练数据，附件3 为测试数据，不可使用测试标签（题目未给）。
- 风化前后类型预测需说明推理依据。
- 亚类划分必须与附件2 的分类逻辑一致。
- 所有定量分析须结合化学成分数据。

## 7. 评分暗示（来自题目描述）
- 模型构建过程的科学性、可解释性。
- 结果表格的规范性与完整性。
- 图表质量与决断的清晰度。
- 对风化影响的量化分析与统计检验。

## 8. 风险与不确定说明
- 部分化学成分名称在不同文献中可能对应不同氧化物，需统一引用标准。
- "风化"定义可能基于肉眼观察，存在一定主观性，题目未提供风化程度的定量指标。
- 附件3 不含颜色和风化标识，推测时须引入合理假设。
- 少量样本成分总和远偏离100%，可能是数据记录错误或存在其他未检测物质，需预处理。
