# 附件清单与质量评估

## 附件1：文物玻璃成分数据
- **文件名**：`附件1.xlsx`（或 `.csv` 备用）
- **类型**：结构化表格
- **可读性**：良好，无扫描版
- **字段概览**：
  - `文物编号`：样本唯一标识（如22、23、…）
  - `风化`：二值（风化、未风化）
  - `颜色`：多类别文本
  - `类型`：高钾、铅钡
  - 化学成分列：约 25 个氧化物百分比（SiO₂, Na₂O, K₂O, CaO, MgO, Al₂O₃, Fe₂O₃, PbO, BaO, …）
- **数据质量**：
  - 存在少量缺失值（空白），部分文物缺乏某些微量成分。
  - 成分百分比合计不等于100%，偏离在 5%-30% 之间，可能与未检测物质或测量误差有关。
  - 颜色标注有缩写（如“浅蓝”“绿”），可能产生歧义。
- **OCR 需求**：无。
- **风险**：风化/未风化标签是否完全可靠？题目未说明风化判定标准，可能影响建模。

## 附件2：玻璃亚类分类信息
- **文件名**：`附件2.xlsx`（或 `.csv`）
- **类型**：结构化描述表
- **可读性**：良好
- **字段概览**：
  - `玻璃类型`：高钾、铅钡
  - `亚类`：各类型的子类别名称（如高钾-A，高钾-B，铅钡-A 等）
  - `特征描述`：化学元素含量范围或定性描述（如“Al₂O₃含量较高”）
- **数据质量**：
  - 部分描述为定性语言（如“较高”“较低”），缺乏精确阈值，需人工或统计手段量化。
- **OCR 需求**：无。
- **风险**：描述含主观性，不同人解读可能产生划分差异。

## 附件3：待鉴别文物成分数据
- **文件名**：`附件3.xlsx`
- **类型**：结构化表格
- **可读性**：良好
- **字段概览**：
  - `文物编号`：待预测样本编号（如1, 2, …）
  - 化学成分列：同附件1
  - **无** “风化”和“颜色”列
- **数据质量**：
  - 完整性好，无缺失值观察。
  - 成分百分比同样存在合计不全的问题。
- **风险**：缺少风化信息与颜色，预测风化前后类型时需依赖额外假设或先验分布。

## 附件总评
所有附件为 Excel 格式，可直接读取程序化处理。缺失值及成分合计偏差需在 EDA 阶段详细分析。
