## v2.4 规则

1. `problem_type` 决定论文段落功能权重。
2. `model_family` 决定模型解释深度和图表优先级。
3. 主模型必须绑定至少 1 张机制/流程图、1 张核心结果图、1 张检验/对比图。
4. baseline 不是装饰，必须进入结果对比或误差检验。
5. 若主模型无法闭合，优先降级到解释力更强、能按时求解的模型。
