# 图文权重与点位分布规则

## 主规则

图表不是装饰，它承担“压缩解释成本”的功能。每张图必须回答一个问题：读者看完这张图，能比只读文字更快理解什么？

## 点位分布

| 位置 | 推荐图表 | 禁止问题 |
|---|---|---|
| 问题分析后 | 总技术路线图、任务拆解图、模型链路图 | 只放空洞流程图 |
| 数据处理后 | 缺失/异常图、变量关系图、指标构造图 | 放太多原始描述图 |
| 模型建立前后 | 机制图、变量关系图、目标约束结构图 | 公式没有图解释 |
| 求解过程 | 算法流程图、收敛图、路径图、可行域图 | 只贴代码截图 |
| 结果分析开头 | 核心结果摘要卡、排序矩阵、预测区间、空间图 | 先大段文字再给图 |
| 检验段 | 残差、敏感性、鲁棒性、消融、对比实验图 | 只说“结果较稳定” |
| 总结前 | 结论矩阵、方案对比图、风险图 | 重复前文结果 |

## 数量规则

- 20 页以内：正文主图 5-7 张，主表 3-6 张。
- 20-30 页：正文主图 7-10 张，主表 4-8 张。
- 30 页以上：正文主图 9-12 张，主表 5-10 张，细节转附录。
- 任一图类占比不得超过 35%。
- 连续两张同类型图必须合并、换图类，或转附录。

## 美观门禁

- 图中文字必须为中文，数字单位必须完整。
- 字号不得低于 8 pt；正文主图建议坐标轴 10-11 pt、图例 9-10 pt、注释 8-9 pt。
- 坐标轴标题、图例、注释不得乱码；必须设置中文字体与 `axes.unicode_minus=False`。
- 禁止直接使用 Matplotlib、Seaborn、Plotly 默认配色。
- 禁止彩虹色。颜色总数不超过 5 个主色。
- 背景保持干净，网格线只做辅助，不喧宾夺主。
- 优先 SVG/PDF，PNG 不低于 300 dpi。
