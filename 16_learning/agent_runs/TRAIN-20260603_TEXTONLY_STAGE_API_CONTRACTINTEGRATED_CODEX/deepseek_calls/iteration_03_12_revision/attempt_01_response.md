Problem Formulation\nGiven a training set of N observations {(R_i, G_i, B_i, C_i)} where C_i is the true concentration, we seek a model f(R,G,B) = Ĉ that minimizes the sum of squared errors. The problem is formulated as a standard linear regression: C = β₀ + β₁·R + β₂·G + β₃·B + ε, where ε ~ N(0, σ²). The objective is to estimate β coefficients that yield the best predictive accuracy on unseen data.\n\n## 3. Model Description\nThe Multiple Linear Regression (MLR) model is defined as:\n\n$$C = \\beta_0 + \\beta_1 \\cdot R + \\beta_2 \\cdot G + \\beta_3 \\cdot B + \\epsilon$$\n\nCoefficients β are estimated using ordinary least squares (OLS) via the Moore-Penrose pseudoinverse.
