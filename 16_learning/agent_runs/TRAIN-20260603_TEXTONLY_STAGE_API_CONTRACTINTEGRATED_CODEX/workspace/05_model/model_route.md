## 4. 模型路由矩阵

### Q1: 关键因素识别与综合评价
- **主模型**: 层次分析法 (AHP) + 熵权法组合赋权 TOPSIS
- **备选模型**: 主成分分析 (PCA) 降维后线性加权
- **数据需求**: 归一化指标矩阵；缺失值需插补；异常值需 Winsorize。
- **核心假设**: 指标间独立可加；权重反映决策者偏好或数据变异性。
- **可解释性**: 高；权重和得分可溯源。
- **复现性**: 高；工具链成熟。
- **验证方案**: 排序稳定性检验（随机扰动权重 +/-10%）、与纯 AHP 或纯熵权结果对比。
- **风险**: 若指标共线性强，组合赋权可能失真；需人工确认判断矩阵一致性。
- **图表建议**: 权重条形图、排序雷达图、敏感性 tornado 图。

### Q2: 目标变量预测
- **主模型**: 集成学习（LightGBM 或 Random Forest）
- **备选模型**: 带正则化的多元线性回归（ElasticNet）
- **数据需求**: 时间序列或横截面特征矩阵，目标变量连续/离散；需划分训练/验证/测试集；需处理缺失值和中度异常值。
- **核心假设**: 历史模式可外推；特征与目标非线性关系可通过树模型捕捉。
- **可解释性**: 中（特征重要性可输出，但个体预测路径不透明）；若可解释性要求高，可降级为线性模型并接受精度损失。
- **复现性**: 中高；固定随机种子可复现；但超参数搜索空间需记录。
- **验证方案**: K-fold 交叉验证、保留测试集上 RMSE/MAE/MAPE、残差诊断、预测区间（分位数回归或 bootstrap）。
- **风险**: 数据泄漏风险（如时间特征提前泄露）；过拟合高风险；外推边界未定；需对特征分布漂移进行监控。
- **图表建议**: 实际 vs 预测散点图、残差分布图、特征重要性排行图。

### Q3: 资源约束下的方案优化
- **主模型**: 多目标线性/非线性规划 (MOLP/MONLP)
- **备选模型**: 遗传算法 (GA) 或粒子群优化 (PSO)
- **数据需求**: 决策变量边界、约束系数矩阵、目标函数系数（可能来自 Q2 预测）；基线方案已知。
- **核心假设**: 线性或可近似；多目标可转化为单目标（如加权和或 ε-约束）。
- **可解释性**: 中；规划模型结构清晰，但多目标权衡需说明。
- **复现性**: 中；确定性求解器可复现，但启发式算法需记录参数。
- **验证方案**: 与基线方案对比、敏感性分析（目标权重、约束右端项）、极端情景测试。
- **风险**: 若真实系统强非线性，规划模型可能次优；启发式求解可能陷入局部最优；多目标权重的主观性。
- **图表建议**: Pareto前沿图、方案对比柱状图、约束松弛敏感性图。
