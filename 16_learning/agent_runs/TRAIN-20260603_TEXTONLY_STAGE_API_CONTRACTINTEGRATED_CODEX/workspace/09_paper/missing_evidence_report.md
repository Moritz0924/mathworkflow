# 缺失证据报告

本文所提出的模型和结论基于模拟数据集，目前存在以下证据缺口：

1. **真实实验数据**：未收集T01、T02、T03真实溶液的RGB图像与对应浓度值，无法评估模型在真实噪声、光照变化、容器散射等条件下的性能。
2. **混合物质样本**：模型仅针对单一物质溶液验证，未测试多组分混合物场景下的预测能力。
3. **成像条件标准化**：未建立标准化的图像采集方案（固定光源、摄像头设置、拍摄距离），环境变量对预测的影响未量化。
4. **图文件实体**：文中引用的图F001-F007目前为占位描述，实际图像文件尚未生成，无法进行视觉检查。
5. **非线性模型对比**：仅比较了PLS，未引入支持向量回归（SVR）、随机森林或神经网络等非线性模型，对线性的评价可能存在偏差。
6. **外部独立验证**：所有结果基于同一生成机制下的训练/测试划分，缺乏来自不同来源的外部独立验证集。
7. **统计检验**：某些结论（如残差正态性）引用了统计检验但仅为模拟数据结果，需真实数据再确认。
8. **超参数敏感性**：岭回归正则化参数λ的选取未进行系统调优，仅采用默认值。

建议在后续工作中针对以上缺口逐项补充实验或论证，必要时调整论文结论的强度。
