# 基于多通道RGB光谱数据的水体污染物浓度预测方法研究

## 摘要
针对传统化学分析方法在实时环境监测中的局限性，本文利用多通道RGB光谱数据，结合多元回归与机器学习方法，构建了一种快速、非接触式的水体污染物浓度预测模型。研究采集了不同浓度标准溶液在受控照明条件下的RGB图像，提取各通道的光谱特征，并采用偏最小二乘回归（PLSR）以及支持向量回归（SVR）进行建模。实验结果表明，所提模型在测试集上的决定系数R²达到0.95以上，平均绝对百分比误差低于5%，能够有效预测硝酸盐、磷酸盐等典型污染物的浓度。该方法为现场快速检测提供了可行的技术方案。

## 关键词
多通道光谱；RGB图像；浓度预测；偏最小二乘回归；水体污染物

## 1. 引言
水体污染问题日益严重，快速准确地检测污染物浓度对于水环境保护至关重要。传统检测方法如分光光度法、离子色谱法虽然精度高，但需要复杂的样品前处理，耗时较长，难以实现现场实时监测。近年来，基于光谱分析的无损检测技术得到广泛应用，其中RGB成像因成本低、操作简便而受到关注。然而，RGB图像仅包含三个颜色通道，如何从有限的光谱信息中有效提取浓度相关特征，并建立稳健的预测模型，是当前研究的关键问题。

本文针对水体中常见污染物，提出了一套基于多通道RGB光谱数据的浓度预测流程。主要贡献包括：(1)设计了一种标准化的图像采集与预处理方法，以减少环境光照对RGB值的影响；(2)结合多变量统计分析与机器学习算法，构建了高精度的浓度预测模型；(3)通过实验验证了模型的有效性和鲁棒性，为实际应用提供了参考。论文结构如下：第2节回顾相关工作，第3节描述数据采集与预处理，第4节介绍所采用的模型与方法，第5节展示实验结果与分析，第6节进行讨论，最后第7节总结全文。

## 2. 相关工作
目前，基于光谱的浓度预测研究主要集中在近红外（NIR）和拉曼光谱。例如，Zhang等人利用NIR光谱结合卷积神经网络预测土壤中的重金属浓度，取得了良好效果。在可见光波段，RGB成像被用于食品、农业等领域，如通过叶片的颜色特征评估氮含量。Li等（2021）采用RGB图像预测水体中藻类浓度，表明可见光特征与生物量存在相关性。但针对水体污染物的RGB预测研究尚不充分，主要挑战在于水体颜色变化微弱，且易受背景干扰。本文在前人工作基础上，针对溶液体系优化了特征提取方法，并对比了多种回归模型的性能，以验证RGB光谱在浓度预测中的潜力。

## 3. 数据与预处理
### 3.1 实验材料与仪器
配制不同浓度的硝酸钠（NaNO₃）和磷酸二氢钾（KH₂PO₄）标准溶液，浓度范围分别为0.1 mg/L至100 mg/L，共20个浓度梯度。使用工业相机（Basler acA1920-155uc）在自制暗箱中拍摄溶液图像，光源为D65标准光源，相机参数固定。每个浓度样本采集10幅图像以评估重复性。

### 3.2 数据预处理
对原始图像进行裁剪，提取中心区域100×100像素的感兴趣区域（ROI）。计算每个通道（R、G、B）的平均像素值及标准差，作为原始特征。为消除光照不均，采用白板校正法，以纯水样本作为参考，计算各个通道的反射率比值。此外，对特征进行标准化处理，使其均值为0，方差为1。

最终数据集包含20个样本，每个样本包含6个特征（R_mean, G_mean, B_mean, R_std, G_std, B_std），以及对应的浓度标签。数据集划分为训练集（14个样本）和测试集（6个样本），确保浓度范围覆盖。

## 4. 模型与方法
### 4.1 偏最小二乘回归（PLSR）
PLSR是一种融合主成分分析与多元回归的方法，通过寻找投影方向使得输入与输出的协方差最大化，有效解决高维共线性问题。给定输入矩阵X和输出向量y，PLSR模型可表示为：
y = TQ' + E
其中T为X的得分矩阵，Q为载荷矩阵。因子数目通过10折交叉验证确定，选用使预测残差平方和（PRESS）最小的因子数。

### 4.2 支持向量回归（SVR）
SVR通过核函数将数据映射到高维特征空间，并在此空间中构建ε-insensitive损失函数的线性回归模型。优化问题为：
min 1/2 ||w||^2 + C ∑(ξ_i + ξ_i^*)
subject to y_i - w·φ(x_i) - b ≤ ε + ξ_i, etc.
选用径向基核函数（RBF），超参数C和γ通过网格搜索结合交叉验证优化，范围C∈[0.1,100], γ∈[0.01,10]。

### 4.3 评估指标
采用决定系数（R²）、均方根误差（RMSE）和平均绝对百分比误差（MAPE）作为评价指标，公式如下：
R² = 1 - SS_res / SS_tot
RMSE = √(1/n ∑(y_true - y_pred)²)
MAPE = 100% / n ∑|(y_true - y_pred)/y_true|

## 5. 实验结果与分析
### 5.1 预测性能对比
表1列出了两种模型在测试集上的性能指标。PLSR在硝酸盐预测中R²为0.962，RMSE为2.35 mg/L，MAPE为4.2%；SVR的R²为0.951，RMSE为2.87 mg/L，MAPE为5.1%。对于磷酸盐，PLSR的R²为0.948，SVR为0.932。整体上PLSR略优于SVR，且计算效率更高，适合在线应用。

### 5.2 特征重要性分析
PLSR模型的回归系数显示，R通道的平均值在所有特征中贡献最大，表明红色波段对浓度变化最为敏感。这与硝酸盐溶液在红光区域的吸收光谱特征一致。B通道的重要性次之，而通道标准差对预测贡献较小。该结果为特征筛选和传感器波段选择提供了依据。

## 6. 讨论
尽管RGB成像仅提供三个宽波段信息，但通过标准化校正和多元回归仍能建立有效的浓度预测模型。然而，本方法存在若干局限性：首先，对光照条件的高度敏感性要求严格的暗箱环境，限制了其户外部署；其次，对于浓度极低（<0.05 mg/L）的样本，预测误差显著增大，可能与相机传感器的灵敏度阈值有关；再次，模型针对特定污染物进行训练，泛化至其他物质需扩展样本库。

与其他高光谱技术相比，RGB方法牺牲了光谱分辨率，但大幅降低了硬件成本和操作复杂度，适用于资源受限的现场场景。未来工作可结合多光谱传感器，或引入迁移学习以提升不同光照和基质下的鲁棒性。此外，探索端到端的卷积神经网络直接从图像预测浓度，可避免人工特征设计的偏差。

## 7. 结论
本文提出了一种基于多通道RGB光谱数据的水体污染物浓度预测方法，利用PLSR和SVR模型实现了对硝酸盐和磷酸盐浓度的准确预测。实验证明，PLSR模型在R²和RMSE上表现更优，且特征分析揭示了R通道的主导作用。该研究为低成本、快速的水质现场检测提供了理论基础和技术途径。后续研究将扩大污染物种类，优化模型泛化能力，并开发便携式检测系统。
