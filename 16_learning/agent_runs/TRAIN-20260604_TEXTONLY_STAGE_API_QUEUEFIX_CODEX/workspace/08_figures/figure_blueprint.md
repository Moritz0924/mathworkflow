# 图表蓝图 (Figures Stage)

## 总览
本阶段共设计8张正式图表，类型覆盖柱状图、热力图、散点图、折线图、预测区间、残差诊断、特征重要性和分组柱状图，服务于模型分析章节。

## 蓝图矩阵
- F001: 综合得分对比（bar）绑定R001；用于模型结果展示。
- F002: 权重热力图（heatmap）绑定R002；展示指标权重。
- F003: Pareto前沿（scatter）绑定R003；优化权衡。
- F004: 敏感性分析（line）绑定R004；参数影响。
- F005: 预测区间（line_interval）绑定R005；验证预测能力。
- F006: 残差诊断（scatter）绑定R006；模型残差检查。
- F007: 特征重要性（barh）绑定R007；特征贡献度。
- F008: 情景对比（group_bar）绑定R008；多情景鲁棒性。

所有图均使用非默认配色，中文标注，矢量SVG格式。
