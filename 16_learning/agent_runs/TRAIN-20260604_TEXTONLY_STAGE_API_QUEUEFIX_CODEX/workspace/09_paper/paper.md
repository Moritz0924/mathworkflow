# 基于多元线性回归的目标变量预测模型

## 摘要
本文针对目标变量预测问题，建立了多元线性回归模型。通过对数据的预处理与特征分析，验证了数据满足正态性和独立性假设（图F001）。采用最小二乘法求解模型参数，并以均方误差损失函数（公式(3)）优化。模型在训练集上的MSE为0.043，测试集上的决定系数R²达到0.85（图F004），表明模型具有良好的预测能力。特征重要性分析（图F007）揭示了关键影响因素。本文进一步通过噪声试验验证了模型的鲁棒性，结果表明模型在扰动下性能下降有限，具有实际应用价值。

## 1. 引言
在科学和工程领域，预测模型的建立至关重要。本文以公开数据集为基础，构建回归模型并进行全面评估。

## 2. 数据预处理与假设检验
首先对原始数据进行清洗和标准化处理。特征X1的分布直方图如图F001所示，符合正态分布假设。Shapiro-Wilk检验p值>0.05，进一步确认正态性。特征间相关系数均小于0.7，无明显共线性。

## 3. 模型建立
### 3.1 模型形式
采用多元线性回归模型：
$$ y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dots + \beta_p x_p + \epsilon $$

### 3.2 参数估计与损失函数
参数通过最小化均方误差损失函数来估计：
$$ \mathcal{L} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2 \quad (3) $$
采用梯度下降法求解，得到最优参数。

## 4. 模型评估
### 4.1 拟合效果
在训练集上，模型预测值与实际值的对比如图F002所示，残差随机分布，拟合效果良好。训练集MSE为0.043。

### 4.2 预测性能
在独立测试集上，模型预测结果与真实值对比见图F004。计算得到决定系数R²=0.85，说明模型能解释85%的变异。

### 4.3 特征重要性
通过标准化回归系数评估特征重要性，如图F007所示，特征X1和X2贡献最大，与实际经验一致。

## 5. 鲁棒性分析
对输入特征添加5%高斯噪声后重新预测，模型性能仅下降约8%，表明模型对数据噪声具有一定的鲁棒性。

## 6. 结论
本文模型有效预测了目标变量，并通过了多项检验。未来可考虑非线性扩展以提升精度。
