## 问题一

分析环境变量和管理变量对单位面积产量的影响方向，识别主要驱动因素，并给出可解释的产量预测模型。

## 问题二

比较至少三类候选模型的预测误差，例如多元线性回归、带正则化的回归、二次响应面模型、树模型或其他合理模型。说明小样本条件下的验证方式和过拟合控制策略。

## 问题三

预测待评估批次 `P01`、`P02`、`P03` 的单位面积产量，给出不确定性说明，并提出可复用的水肥调控建议。
