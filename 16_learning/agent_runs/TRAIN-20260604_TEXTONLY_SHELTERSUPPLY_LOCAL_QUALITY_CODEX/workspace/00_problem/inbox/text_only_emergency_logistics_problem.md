## 问题一

建立可解释的配送路径基线模型，说明距离、需求量、时间窗和道路可靠度如何进入目标函数与约束。

## 问题二

比较至少三类候选模型，例如最近仓库贪心、节约里程启发式、鲁棒路径优化或其他合理模型。给出模型选择依据，并说明如何验证路径可行性。

## 问题三

给出推荐配送方案、车辆数量、路径顺序和应急扰动下的调整策略。至少分析道路可靠度下降、需求量上升、时间窗压缩三类情景。
