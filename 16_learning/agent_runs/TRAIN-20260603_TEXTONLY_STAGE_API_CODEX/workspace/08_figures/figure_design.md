## 图表类型选择依据
- 统计评价：权重热力图直观展示指标重要性矩阵。
- 排序对比：水平柱状图清晰呈现城市得分排序。
- 敏感性分析：热力图展示参数扰动对排名的影响模式。

## 配色方案
- 热力图：teal_orange_diverging（#008080 至 #E07A5F），中点为白色。
- 柱状图：muted_blue_gold（柱体 #3A7CA5，边框 #D4A373）。
- 字体：SimHei，保证中文渲染。
- 背景：白色，无网格线干扰。

## 导出要求
- 优先 SVG（矢量），同时生成 PNG 备用。
- 分辨率 ≥150 dpi。
- 图中所有文本为中文（指标名、坐标轴标题、图例等）。

## 合同绑定
每张正式图进入 figure_contract.csv，绑定 result_id 或 evidence_source，填写 used_in_section 和 latex_label。
