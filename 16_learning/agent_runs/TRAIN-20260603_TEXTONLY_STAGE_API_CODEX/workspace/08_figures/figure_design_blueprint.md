## 2. 图表类型选择依据

- **层级图**：适合展示指标体系的内在结构，避免大段文字。
- **热力图**：直观对比多城市多指标得分，识别空间模式和异常值。
- **Pareto前沿图**：多目标优化中展示成本-效果权衡，帮助决策者选择非支配解。
- **龙卷风图**：敏感性分析中表现各参数对排名的相对影响，方向与幅度一目了然。
- **预测区间图**：同时呈现点预测和不确定性区间，增强模型可信度。

## 3. 图表去冲突

- 每张图服务于不同章节和论证点，避免重复。
- 热力图（FIG002）与Pareto图（FIG003）虽涉及评价，但前者强调空间分布，后者强调解集特征。
- 龙卷风图（FIG004）补充权重稳健性，不与预测图（FIG005）冲突。

## 4. 配色与风格

- 统一采用 muted_blue_gold 和 teal_orange_diverging 配色策略。
- 热力图使用红-蓝渐变，Pareto图蓝-橙区分支配状态。
- 字体统一为 SimHei 或无衬线中文字体，保证渲染一致性。
- 所有图表输出为 SVG 矢量格式，确保缩放质量。
