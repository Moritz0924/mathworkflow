# Q1 Code Implementation

## Model
Linear regression baseline for predicting drought resistance from plant traits.

## Dependencies
- Python 3.8+
- pandas >= 1.2
- numpy >= 1.20
- scikit-learn >= 0.24

## Usage
```bash
cd 06_code/Q1
python solve_q1.py
```

## Input
- `03_data/drought_data.csv` (or synthetic data generated if missing)

## Output
- `07_results/q1_candidate_predictions.csv`
- `10_ai_logs/run_q1.log`

42 (for reproducibility)
