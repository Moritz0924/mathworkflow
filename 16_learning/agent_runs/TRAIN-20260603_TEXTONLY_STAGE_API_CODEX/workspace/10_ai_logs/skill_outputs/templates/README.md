# 技能输出模板

任意技能调用都可以使用 `generic_input_manifest.yaml` 和 `generic_promotion_notes.md`。如果输出类别有额外绑定或校验规则，则使用 `by_category/<category>/`。

类别：

```text
literature_search
citation_verification
data_reproducibility
prior_reading
figure_design
section_writing
review_response
language_polish
post_final_presentation
```

模板规则：

```text
input_manifest.yaml 说明读取了什么，以及本次调用是否被允许。
promotion_notes.md 说明归档输出能否被下游使用。
```
