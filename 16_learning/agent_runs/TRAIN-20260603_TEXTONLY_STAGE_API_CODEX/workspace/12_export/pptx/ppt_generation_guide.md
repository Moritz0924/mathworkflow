# 展示材料（PPT）生成说明

由于本次沙盒训练未收到明确的 PPT 生成需求，本阶段只提供生成指导，不生成实际演示文稿。

## 何时需要生成 PPT
- 当人工最终闸门确认后，若需要面向评审的展示材料，可使用 `nature-paper2ppt` 技能基于已批准论文生成幻灯片。

## 生成前提
- 最终论文 PDF 或 LaTeX 源码可用
- 所有图表文件可用
- 人工确认已做出

## 生成原则
- 幻灯片内容**仅限于**已批准论文中呈现的问题、模型、结果、图表、结论和局限，**不得新增**任何结果或图表论断。
- 所有文字（标题、正文、注释、图表标注）为中文，英文缩写和变量名除外。
- 每张幻灯片应可追溯到论文的特定章节或图表编号。

## 推荐操作
1. 运行 `nature-paper2ppt` 技能，输入论文摘要和关键图表。
2. 导出 PPTX 文件至 `12_export/pptx/`。
3. 人工审核确保无新增事实、全中文。

## 注意事项
- 若论文中使用了第三方数据或图表，转载时须注明版权来源。
- 幻灯片风格应简洁，避免过度动画。
