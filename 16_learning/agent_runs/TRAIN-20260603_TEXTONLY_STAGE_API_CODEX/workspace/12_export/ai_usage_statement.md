# AI 使用声明

本文档说明在本项目数学建模工作流中使用的 AI 工具及其角色。

## 使用的 AI 模型/服务
- Codex 执行器（语言模型），作为工作流内的任务执行代理，严格按阶段指令和合同总线规则运行。
- `nature-skills`（专家工具集），用于特定阶段任务（如文献检索、数据获取、图表导出、论文润色等），不替代工作流控制器。

## AI 使用范围
- **文献检索**：`nature-academic-search` 和 `nature-citation` 用于查找与验证引用。
- **数据处理**：`nature-data` 辅助数据来源说明，未替代人工判断。
- **模型构建**：AI 仅参与模型路线建议，核心建模决策和验证由人类完成。
- **代码生成**：代码由 Agent 生成，通过人工审核与验证。
- **图表生成**：`nature-figure` 生成符合规范的科学图表，所有数据来自合同登记的结果。
- **论文撰写**：`nature-writing` 协助草拟章节，全文经过人类审阅和多轮修订。
- **审稿与修订**：`nature-response` 风格逻辑用于评论归并，修订决策由人类确认。
- **润色**：`nature-polishing` 仅进行语言润色，未改变数字、公式、标签或论证。
- **展示材料**：如有 PPT 生成，仅重组已批准论文内容。

## 人工监督
- 所有阶段均由人工闸门控制，最终交付物经过人工最终确认。
- 合同总线确保 AI 产出可追溯、可验证。
- 本声明随提交包一同提供。

## 责任声明
本项目最终内容由人类作者负责。AI 工具仅作为辅助手段，其输出已通过多级验证与人工审核。
