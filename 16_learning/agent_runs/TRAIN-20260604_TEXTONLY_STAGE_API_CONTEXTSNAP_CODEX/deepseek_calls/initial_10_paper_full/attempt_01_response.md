颜色通道；物质浓度；多元线性回归；模型诊断；不确定性量化\n\n## 1 问题重述与分析\n\n某检测设备对溶液进行拍摄与预处理后，输出红（R）、绿（G）、蓝（B）三个归一化颜色通道。现提供10个标准样本的真实浓度 \\(C\\) 与颜色读数（表1），以及3个仅有颜色读数而无浓度的待测样本 T01、T02、T03。要求：\n- **问题一**：分析RGB与浓度的关系，建立可解释的浓度预测模型，并说明各通道的作用方向。\n- **问题二**：比较至少三类候选模型（如线性回归、多项式回归、集成回归等），给出模型选择依据，并说明控制小样本过拟合风险的方法。\n- **问题三**：预测待测样本浓度，提供不确定性说明，并提出可复用的检测流程建议。\n\n由于标准样本数仅10个，特征维度为3，属于典型的小样本建模问题。同时，从标准样本的单调变化趋势（浓度增大时R递减、G和B递增）可初步判断存在强相关关系，适合优先尝试线性模型并在其基础上讨论过拟合的控制策略。以下章节依次展开。\n\n## 2 模型建立\n\n### 2.1 数据探索\n\n为直观把握变量间的关联，绘制浓度 \\(C\\) 与 R、G、B 的三通道散点图矩阵（图F005）及Pearson相关系数热力图（图F006）。热力图显示R与浓度呈强负相关（约 -0.99），G、B与浓度呈强正相关（约0.99），且R、G、B之间亦高度线性相依。该结构表明线性参数化是合理的起点，但也提示可能存在共线性，需要在系数解释时审慎对待。\n\n### 2.2 线性回归模型（问题一）\n\n#### 模型形式\n\n设浓度 \\(C\\) 由颜色通道的线性组合与随机误差构成：\n\n\\[\nC = \\beta_0 + \\beta_1 R + \\beta_2 G + \\beta_3 B + \\varepsilon, \\tag{1}\n\\]\n\n其中 \\(\\varepsilon \\sim \\mathcal{N}(0,\\sigma^2)\\) 且满足独立、同方差假设。利用最小二乘法（OLS）估计系数 \\(\\hat{\\beta}_j\\)。\n\n#### 系数估计与解释\n\nOLS估计结果（冻结自 \\(\\texttt{07\\_results/q1\\_coefficients.csv}\\)）为：\n\n\\[\n\\begin{aligned}\n\\hat{\\beta}_0 &= 1.0672, \\\\\n\\hat{\\beta}_1 &= -1.8214 \\quad (\\text{R通道}), \\\\\n\\hat{\\beta}_2 &= 0.9956  \\quad (\\text{G通道}), \\\\\n\\hat{\\beta}_3 &= 5.6310  \\quad (\\text{B通道}).\n\\end{aligned}\n\\]\n\n各系数的作用方向和量级在图F001中以条形图展示。\n\n- **R通道**：系数为负，表明在其他通道保持不变时，R值上升会导致预测浓度下降。这符合物理直觉——溶液颜色随浓度增大而变深，R分量减小。\n- **G通道**：系数为正但量级较小（约0.996），其单变量影响弱于B通道。\n- **B通道**：系数正值最大，说明B通道是主导正向预测因子，反映出蓝色分量对浓度变化最为敏感。\n\n#### 模型整体表现\n\n模型的确定系数 \\(R^2 = 0.9977\\)，均方根误差 \\(\\text{RMSE} = 0.0674\\ \\text{mg/L}\\)（\\(R05, R06\\)）。该值在标准样本浓度范围（0.5 – 5.0 mg/L）内表明拟合极好，但需警惕样本量小可能带来的过度乐观。\n\n### 2.3 候选模型比较框架（问题二）\n\n为满足问题二要求，引入以下三类候选模型：\n\n1. **多元线性回归**（公式(1)）。\n2. **二次多项式回归**，包括平方项和交互项：\n\n\\[\nC = \\beta_0 + \\sum_i \\beta_i X_i + \\sum_i \\beta_{ii} X_i^2 + \\sum_{i<j} \\beta_{ij} X_i X_j + \\varepsilon, \\quad X_1 = R, X_2 = G, X_3 = B. \\tag{2}\n\\]\n\n3. **支持向量回归（SVR）** 或 **随机森林回归**：通过核方法或集成学习捕捉可能的非线性，但需要更严格的超参数调优以避免过拟合。\n\n**评估策略**：考虑样本量极小（n=10），采用留一交叉验证（LOOCV）计算RMSE作为主指标，辅以平均绝对误差（MAE）。所有模型共享统一的数据划分和随机种子，确保可比。\n\n**过拟合控制**：对于多项式模型，加入L2正则化（Ridge）；对于SVR和随机森林，通过网格搜索确定最优超参数组合，并在LOOCV下评估泛化误差。\n\n> **缺口说明**：因上游 \\(\\texttt{codegen}\\) 与 \\(\\texttt{results\\_freeze}\\) 阶段尚未完成，上述候选模型的训练、比较结果及过拟合控制实验尚未在结果合同中登记。本节目前仅呈现方法学框架，定量对比将在后续修订中补充。\n\n### 2.4 浓度预测与不确定性说明（问题三）\n\n#### 可复用检测流程\n\n1. 使用标准样本建立模型，并通过留一验证评估误差水平。\n2. 对待测样本采集颜色读数，输入模型得到预测值及Bootstrap区间。\n3. 当新批次的标准样本加入后，更新模型并进行诊断，确保质量控制。\n\n> **缺口说明**：因Bootstrap模拟结果尚未冻结，待测样本的最终预测值及不确定性区间仍待上游阶段完成。\n\n## 3 结果与讨论\n\n### 3.1
