同问题一的标准样本数据。\n  - 问题一中获得的关系理解（作用方向、线性程度）可作为先验知识，但不强制作为唯一路径。\n  - 各候选模型的超参数设置（例如多项式的阶数、正则化系数、树的数量等）。\n\n- **输出**\n  - 三种以上模型的训练与评估结果汇总表，包括：训练集拟合指标（R²、RMSE）、留一法/交叉验证（LOOCV 或 K 折）的误差（MAE、RMSE）、模型复杂度度量（参数个数）。\n  - 待测样本的颜色通道数据。\n  - 问题二中选定的模型及其训练好的参数。\n  - 标准样本数据用于不确定性估计（例如残差分布、Bootstrap）。\n\n- **输出**\n  - 待测样本预测浓度表，包含点估计和不确定性度量（如95%置信区间或预测区间）。\n  - 不确定性分析方法说明：假设、方法、适用性。\n  - 可复用检测流程：描述标准样本准备、模型训练、验证、预测的步骤，以及注意事项（如颜色读数稳定性、浓度范围外推限制等）。\n\n- **约束与评价**\n  -
