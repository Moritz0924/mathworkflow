# Symbols and Notation

## Variables
| Symbol | Description | Unit / Range |
|--------|-------------|--------------|
| C | Substance concentration | mg/L |
| R | Normalized red channel intensity | [0,1] |
| G | Normalized green channel intensity | [0,1] |
| B | Normalized blue channel intensity | [0,1] |
| n | Number of standard samples | 10 |

## Model Parameters (Linear Regression)
| Symbol | Description |
|--------|-------------|
| β₀ | Intercept |
| β₁ | Coefficient for R |
| β₂ | Coefficient for G |
| β₃ | Coefficient for B |
| ε | Error term ~ N(0, σ²) |

## Model Performance Metrics
| Symbol | Description |
|--------|-------------|
| RMSE | Root Mean Square Error |
| R² | Coefficient of determination |
| Adj. R² | Adjusted R² |
| MAE | Mean Absolute Error |
| VIF | Variance Inflation Factor |

## Additional Symbols for Polynomial Model
| Symbol | Description |
|--------|-------------|
| βᵢⱼ | Coefficient for interaction or squared terms |
| d | Polynomial degree |

## Uncertainty
| Symbol | Description |
|--------|-------------|
| SE_pred | Standard error of prediction |
| PI | Prediction interval |
| CI | Confidence interval |
