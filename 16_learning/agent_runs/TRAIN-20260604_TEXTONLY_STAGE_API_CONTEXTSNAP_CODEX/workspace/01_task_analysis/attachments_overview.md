## 字段概览
### 标准样本表字段
| 字段 | 说明 |
|------|------|
| sample_id | 样本编号（S01–S10）|

| R | 归一化红色通道值 |
| G | 归一化绿色通道值 |
| B | 归一化蓝色通道值 |

| 字段 | 说明 |

| sample_id | 样本编号（T01–T03）|
| R / G / B | 归一化颜色通道值 |
