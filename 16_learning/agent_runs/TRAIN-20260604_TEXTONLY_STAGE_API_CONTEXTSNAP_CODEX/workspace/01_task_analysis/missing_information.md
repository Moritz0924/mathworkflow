# 缺失信息与歧义记录

## 1. 数据归一化方法未说明
- **问题**：R、G、B 通道的归一化方式（例如除以 255？传感器原始范围？最大最小值归一化？）未提及。
- **影响**：若归一化方法不同，模型表达式中的系数解释会不同；不影响预测数值，但影响可解释性。
- **建议**：人工查阅原始题库 PDF 或通过物理常识假设（常见为 0-1 归一化），并在论文中声明假设。

## 2. 物质化学属性未知
- **问题**：题目未说明待测物质是什么。
- **影响**：不影响数学模型本身，但可能影响“检测流程建议”中对仪器、波长、干扰因素等的具体描述。
- **建议**：模型中提及“某物质”，论文中说明模型适用于该检测设备的颜色响应规律，不承诺跨物质泛化。

## 3. 模型比较的误差指标未指定
- **问题**：问题二要求比较模型预测误差，但未限定具体指标。
- **影响**：可能需选择 RMSE、MAE、R² 等，不同指标可能导向不同最优模型。
- **建议**：选择多重指标（如 RMSE + MAE + R²）并说明各指标侧重，避免单一指标偏差。

## 4. 小样本过拟合控制方法边界不明确
- **问题**：题面仅要求“说明如何控制小样本过拟合风险”，未指定需要实现何种具体方法。
- **影响**：可解释为仅需理论论述，也可解释为需实际应用交叉验证或正则化并展示效果。
- **建议**：在任务分析阶段保留两种解释，由人类确认目标深度。

## 5. 待测样本不确定性类型未指定
- **问题**：问题三要求“给出不确定性说明”，未指定是预测区间、置信区间还是标准差。
- **影响**：不同不确定性度量的计算方法和解释不同。
- **建议**：建议提供预测值的点估计、标准误差以及 95% 预测区间（若模型允许）。

## 6. 检测流程建议的详细程度
- **问题**：“可复用的检测流程建议”可能期望包括仪器操作、浓度范围、模型使用步骤等，但未具体说明。
- **影响**：可能被理解为仅给出模型使用流程，而非完整实验流程。
- **建议**：在论文中先给出模型应用流程，如有必要再补充实验建议，需与人类确认。

## 7. 题库 PDF 中是否存在额外说明
- **问题**：当前沙盒题面可能为题库 PDF 的摘要，原始 PDF 可能包含更多背景、示意图或附加数据。
- **影响**：若存在缺失的原始信息，可能导致论文与官方题面不符。
- **建议**：人工核验题库 PDF 内容，确认无遗漏。

## 总结
以上问题均不影响 intake 阶段的完成，但需在 task_analysis 或模型设计阶段前由人类确认。当前所有不确定信息已记录，未进入题面事实。
