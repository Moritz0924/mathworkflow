# 图表图注银行

用于论文中引用的图表图注，基于合同绑定证据生成。

## F001 OLS系数估计值与方向

**图F001** 多元线性回归模型各变量系数估计值条形图。图中展示截距及R、G、B通道的OLS系数，反映各颜色通道对浓度预测的作用方向和大小（正值为正向影响，负值为负向影响）。数据来源：result R01–R04。

## F002 预测浓度与真实浓度对比

**图F002** 模型预测浓度与标准样本真实浓度对比散点图。对角线附近分布表明模型预测值与真实值高度一致，支持模型拟合优度（R²=0.9977）。数据来源：result R05–R06。

## F003 残差-拟合值图

**图F003** 残差对拟合值的散点图，用于检验同方差性假设。图中未出现明显的“喇叭口”或系统模式，初步支持OLS假设。数据来源：残差计算。

## F004 残差Q-Q图

**图F004** 残差分位数-正态理论分位数图，检验残差正态性。散点大致沿对角线分布，提示残差近似正态。鉴于样本量仅10个，正态性检验效力有限。数据来源：残差计算。

## F005 浓度与各颜色通道散点矩阵

**图F005** 浓度与R、G、B三个通道的散点图矩阵，直观展示线性趋势：浓度与R负相关，与G、B正相关。数据来源：标准样本数据。

## F006 变量相关性热力图

**图F006** 浓度及RGB通道之间的Pearson相关系数热力图，颜色深浅代表相关强度与方向。热力图显示浓度与B及G呈强正相关，与R呈强负相关。数据来源：标准样本数据。

## F007 三种模型留一交叉验证RMSE对比

**图F007** 三种候选模型的留一交叉验证RMSE对比条形图。线性回归（OLS）的CV RMSE为0.0748 mg/L，远低于多项式回归（0.347 mg/L），验证了线性模型在小样本下泛化能力最强。岭回归的CV RMSE为0.098 mg/L，介于两者之间。数据来源：result R07–R09。

## F008 待测样本预测浓度及95%预测区间

**图F008** T01、T02、T03的点预测浓度及其95%预测区间。点估计分别为1.61、3.17、4.38 mg/L，区间宽度约0.52 mg/L，反映基于10点小样本预测的不确定性。数据来源：result R10–R18。
