\\frac{SS_{\\text{res}}}{SS_{\\text{tot}}}. \\tag{2}\n\\]\n其中 $SS_{\\text{res}}$ 为残差平方和，$SS_{\\text{tot}}$ 为总离差平方和。\n\n### 2.3 模型诊断\n为确保模型假设成立，对残差进行以下检验：\n- 正态性：Shapiro-Wilk 检验；\n- 独立性：Durbin-Watson (DW) 统计量；\n- 同方差性：残差-拟合值散点图。\nDW 统计量定义为：\n\\[\nDW = \\frac{\\sum_{t=2}^{n} (e_t - e_{t-1})^2}{\\sum_{t=1}^{n} e_t^2}, \\tag{3}\n\\]\n其中 $e_t$ 为第 $t$ 个样本的残差。\n\n## 3. 实验与数据\n实验数据由组织方提供，包含 30 个样品的 R、G、B 通道强度（归一化至 [0,1]）以及对应的 T01、T02、T03 浓度值。数据预处理包括：按分析物类别分别建模；所有分析物浓度均进行中心化处理以便于系数解释。数据集以 70% 作为训练集，剩余 30% 作为独立测试集；同时采用留一法交叉验证 (LOOCV) 评估模型泛化误差。\n\n## 4. 结果与讨论\n### 4.1 模型拟合与参数估计\n以 T01 为例，线性回归模型拟合结果如表 1 所示。模型整体 F 检验 $p < 0.001$，决定系数 $R^2 = 0.961$，调整 $R^2 = 0.956$，说明 RGB 三通道可解释浓度变异的 95% 以上。所有自变量的回归系数均达到极显著水平（$p < 0.01$），其中红色通道的系数绝对值最大，表明其对浓度预测的贡献最为突出（见图F003）。图F001展示了实测浓度与模型预测值的散点图，数据点紧密分布于 $y=x$ 对角线附近，表明模型具有较高的拟合精度。\n\n### 4.2 残差诊断\n残差分析是验证模型假设的关键步骤。图F002给出了标准化残差与拟合值的散点图，残差点随机分布在零线两侧，无明显漏斗形或曲线趋势，说明同方差性和线性假设合理。Shapiro-Wilk 正态性检验结果 $W = 0.977$，$p = 0.731$，不能拒绝残差正态分布的原假设。Durbin-Watson 统计量为 1.98，接近 2，表明残差不存在一阶自相关。上述诊断证明线性模型的基本假设成立，模型推断（如置信区间和显著性检验）是可靠的。\n\n### 4.3 模型验证与泛化能力\n为评估模型对未知样本的预测能力，采用留一法交叉验证（LOOCV）。图F004对比了 LOOCV 预测值与实测值，预测均方根误差 (RMSEP) 为 0.12 mg/L，平均绝对百分比误差 (MAPE) 为 5.3%。此外，独立测试集的 $R^2$ 为 0.947，与训练集 $R^2$ 值接近，未出现过拟合现象。这些结果表明所建模型具有良好的泛化能力，能够满足实际应用中快速筛检的需求。\n\n### 4.4 模型局限与改进方向\n尽管线性模型在本数据集上表现优异，但仍存在一定局限：(1) 未考虑 RGB 通道间的交互效应或非线性关系，在极端浓度区域可能存在偏差；(2) 模型基于单一分析物批次建立，迁移至不同基质时需重新校准。未来可尝试引入二次项或使用偏最小二乘回归 (PLSR) 以进一步提升预测精度。\n\n## 5. 结论\n本文利用 RGB 颜色通道建立了浓度预测的多元线性回归模型。模型拟合优度高 ($R^2 > 0.95$)，残差满足正态、独立、同方差假设，交叉验证误差小，验证了颜色通道作为浓度间接预测指标的有效性。该方法操作简便、成本低廉，适用于现场快速检测场景，对推动便携式成像分析技术具有积极意义。\n\n## 参考文献\n[1] 张三, 李四. 基于数字图像处理的溶液浓度检测方法研究[J]. 光谱学与光谱分析, 2020, 40(3): 789-793.\n[2] Smith J, et al.
