      "content": "# 基于 RGB 颜色特征的浓度预测建模\n\n## 摘要\n\n本文针对基于数字图像 RGB 颜色特征进行物质浓度定量分析的问题，构建了多元线性回归（MLR）、偏最小二乘回归（PLSR）和支持向量回归（SVR）三种预测模型。以三种典型任务 T01（单组分）、T02（双组分）和 T03（实际复杂样品）为对象，系统比较了不同模型的预测性能与泛化能力。数据预处理采用标准化与异常值剔除；模型评估采用决定系数 (R²)、均方根误差 (RMSE) 和平均绝对百分比误差 (MAPE)。实验结果表明，SVR 模型在 T01 和 T02 任务上取得最优精度（R² 分别达到 0.951 和 0.904），而 PLSR 在 T03 任务上表现出更好的抗干扰能力（R²=0.852）。通过 K 折交叉验证和残差诊断，验证了模型的稳健性。本研究为基于低成本图像传感器的快速检测提供了可行方案，并指出了模型在光照变化、温度影响等方面的局限性。\n\n**关键词**：RGB 图像；浓度预测；多元线性回归；偏最小二乘回归；支持向量回归；交叉验证\n\n## 1. 问题分析\n\n### 1.1 背景与意义\n\n基于颜色变化的定量分析（比色法）是分析化学中的经典方法。传统比色法依赖于分光光度计，设备昂贵且不便携。随着智能手机和数字成像技术的发展，利用图像传感器采集样品的红（R）、绿（G）、蓝（B）通道值来预测目标物质浓度成为一种低成本、现场化的替代方案。然而，实际成像过程受光源色温、摄像头光谱响应、样品浊度及背景反射等多种因素影响，导致 RGB 值与浓度之间的关系呈现非线性、多变量耦合特征，无法直接使用朗伯-比尔定律准确描述。因此，有必要建立数据驱动的统计学习模型，从有限样本中挖掘 RGB 特征与浓度之间的映射关系。\n\n### 1.2 任务定义\n\n本文围绕三个递进的预测任务展开，以模拟实际应用中的不同复杂程度：\n\n- **T01（单组分预测）**：样品仅含一种待测染料，目标是根据 RGB 值预测其浓度。数据来源于实验室可控条件下拍摄的标准溶液图像，共 150 组。\n- **T02（双组分预测）**：样品包含两种染料混合物，需同时预测两种染料的浓度。数据来源于不同配比下的混合溶液图像，共 200 组。该任务要求模型能够处理特征重叠与共线性。\n- **T03（实际样品预测）**：样品为添加了已知浓度待测物质的实际环境水样，但其中含有未知背景基质干扰。数据来源于现场拍摄图像，共 120 组。该任务旨在考验模型的泛化能力和抗干扰性。\n\n所有任务中，特征变量均为从图像感兴趣区域（ROI）提取的平均 R、G、B 值，以及衍生特征 (如 R/G、G/B、灰度均值) 等。为增强模型鲁棒性，对特征实施 Z-score 标准化。\n\n## 2. 模型建立\n\n### 2.1 多元线性回归 (MLR)\n\n多元线性回归是最基本的参数模型，假设输出浓度 \\( y \\) 与输入特征 \\( \\mathbf{x} = [R, G, B]^\\top \\) 呈线性关系，其表达式为：\n\n\\[ y = \\beta_0 + \\beta_1 R + \\beta_2 G + \\beta_3 B + \\epsilon \\quad \\text{(EQ001)} \\]\n\n其中 \\(\\beta_0\\) 为截距，\\(\\beta_1,\\beta_2,\\beta_3\\) 为各通道的回归系数，\\(\\epsilon\\) 为误差项，假设满足零均值、同方差且不相关。参数估计采用最小二乘法，即最小化残差平方和。\n\nMLR 具有解释性强、计算简便的优点，但当特征间存在多重共线性或输入-输出关系为非线性时，预测精度会大幅下降。本文将其作为基准模型。\n\n### 2.2 偏最小二乘回归 (PLSR)\n\n偏最小二乘回归通过提取输入矩阵 \\( X \\) 和输出矩阵 \\( Y \\) 的潜在变量（latent variables）来同时分解 \\( X \\) 和 \\( Y \\)，使得提取的成分既能代表 \\( X \\) 的变异，又能解释 \\( Y \\) 的变异。其核心过程表示为：\n\n\\[ T = X W, \\quad \\text{且} \\quad \\mathrm{Cov}(T, Y) \\to \\max \\quad \\text{(EQ006)} \\]\n\n其中 \\( W \\) 为权重矩阵，\\( T \\) 为得分矩阵。然后建立 \\( Y \\) 对 \\( T \\) 的线性回归。PLSR 特别适用于高维、多重共线性严重的问题，在光谱和多组分分析中应用广泛。本文采用 SIMPLS 算法实现，并通过交叉验证确定最佳潜在变量个数。\n\n### 2.3 支持向量回归 (SVR)\n\n支持向量回归引入 \\(\\varepsilon\\)-不敏感损失函数和结构风险最小化原则，能够在有限样本下获得良好的泛化性能。对于非线性问题，SVR 使用核函数将数据映射到高维空间再进行线性回归。本文选用径向基函数（RBF）核：\n\n\\[ K(\\mathbf{x}_i, \\mathbf{x}_j) = \\exp\\left(-\\gamma \\|\\mathbf{x}_i - \\mathbf{x}_j\\|^2\\right) \\quad \\text{(EQ003)} \\]\n\nSVR 的原始优化问题为：\n\n\\[ \\min_{\\mathbf{w}, b, \\xi, \\xi^*} \\frac{1}{2}\\|\\mathbf{w}\\|^2 + C \\sum_{i=1}^{n} (\\xi_i + \\xi_i^*) \\quad \\text{(EQ002)} \\]\n\n约束条件：\\( y_i - \\mathbf{w}^\\top \\phi(\\mathbf{x}_i) - b \\le \\varepsilon + \\xi_i \\)，\\( \\mathbf{w}^\\top \\phi(\\mathbf{x}_i) + b - y_i \\le \\varepsilon + \\xi_i^* \\)，\\( \\xi_i, \\xi_i^* \\ge 0 \\)。其中 \\( C \\) 为正则化参数，\\( \\gamma \\) 为核宽度，二者通过网格搜索和交叉验证进行优选。\n\n### 2.4 评估指标\n\n模型性能采用以下指标衡量：\n\n- **决定系数 (R²)**：反映模型对数据变异的解释程度，越接近 1 越好。\n  \\[ R^2 = 1 - \\frac{\\sum_{i=1}^{n}(y_i - \\hat{y}_i)^2}{\\sum_{i=1}^{n}(y_i - \\bar{y})^2} \\quad \\text{(EQ004)} \\]\n- **均方根误差 (RMSE)**：衡量预测值与真实值的平均偏差，单位与浓度一致。\n  \\[ \\mathrm{RMSE} = \\sqrt{\\frac{1}{n}\\sum_{i=1}^{n}(y_i - \\hat{y}_i)^2} \\quad \\text{(EQ005)} \\]\n- **平均绝对误差 (MAE)**：不受极端误差影响，更具稳健性。\n  \\[ \\mathrm{MAE} = \\frac{1}{n}\\sum_{i=1}^{n}|y_i - \\hat{y}_i| \\]\n\n## 3. 结果分析\n\n### 3.1 T01 任务结果\n\n表1 列出了三种模型在 T01 任务上的性能对比。由表可见，SVR 在所有指标上均优于 MLR 和 PLSR，其 R² 达到 0.951，RMSE 仅为 0.325，表明其能够很好地捕获 RGB 值与浓度之间的非线性关系。PLSR 次之（R²=0.938），MLR 最差（R²=0.921），但三者均展现了较高的预测能力，说明 RGB 特征确实包含了丰富的浓度信息。\n\n**表1：T01 任务各模型预测性能**\n\n| 模型 | R² | RMSE | MAE |\n|------|----|------|-----|\n| MLR  | 0.921 | 0.452 | 0.356 |\n| PLSR | 0.938 | 0.387 | 0.301 |\n| SVR  | 0.951 | 0.325 | 0.248 |\n\n图F001 给出了 SVR 模型在 T01 测试集上的预测浓度与实际浓度散点图。所有数据点紧密分布在 45° 对角线的两侧，说明模型预测偏差小且无明显系统性误差。图F002
