# 缺失证据报告 (paper_full 阶段)

## 1. 图形与可视化缺失
- **T03 预测散点图**：未提供 T03 任务下各模型的预测 vs 实际浓度图，无法直观比较 PLSR 与 SVR 的分布差异。
- **交叉验证箱线图**：仅以表格形式报告了5折交叉验证结果，缺少箱线图或误差条图以展示各折性能的波动。
- **参数敏感性热图**：SVR 超参数 C 和 γ 的联合影响仅用文字说明，未用热图或等高线图可视化。

## 2. 数据与实验限制
- **外部验证缺失**：所有结果均基于同一数据集的随机划分，未使用独立的外部数据集验证模型泛化能力。
- **环境鲁棒性测试缺失**：未包含不同光照强度、拍摄角度、摄像头型号下的性能比较实验。
- **样本量偏小**：各任务样本量在120~200之间，对于统计学习模型偏小，可能限制结论的普适性。

## 3. 文献与引用
- 所有引用均为占位虚构文献，未经学术数据库核验，需人工替换为真实文献。

## 4. 合同与追溯
- 论断 C03（T03 任务中 PLSR 优于 SVR）目前仅基于表格数值比较，缺少专项图形和统计检验支持，证据等级评估为 medium。
- 论断 C07（综合结论）为合成性论断，依赖多条证据链，但其中部分子证据（如 T03 可视化）缺失，可能导致结论强度被高估。

## 建议
- 在进入 auto_review 或 final_export 之前，至少补充 T03 散点图和交叉验证箱线图。
- 对于无法立即补充的证据，应在论文正文中明确标注为“初步结果”或“待验证”，并相应降低结论的肯定程度。
