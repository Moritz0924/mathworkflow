# 模型族路由方案

> 基于 `01_task_analysis/task_decomposition.md` 和 `03_data/data_quality_report.md` 制定。

## 1. Q1：城市交通拥堵综合评价

### 1.1 数据适配性
- 15 个候选评价指标，数据缺失率 <2%，异常值已处理。
- 指标量纲不一，需标准化；指标间相关性中等。

### 1.2 主模型：熵权-TOPSIS
- **模型族**：统计评价（客观赋权 + 多属性决策）
- **输入**：标准化指标矩阵 `X_n×m`
- **输出**：各区域拥堵综合得分与排名
- **核心公式**：信息熵（F001）、熵权（F002）、加权标准化（F003）、贴近度（F004）
- **关键假设**：样本独立；权重仅由数据信息量确定。
- **复现成本**：低（numpy/pandas）
- **失败后备选**：若熵值极端，改 CRITIC 法或等权法；若需定性，改模糊综合评价（Q1_FB1）。

### 1.3 备选模型
- 模糊综合评价（处理定性指标）
- CRITIC 权重法（权重对比）

### 1.4 验证方案
- 权重扰动 ±10%，Spearman 相关系数应 >0.9。
- 与等权法、CRITIC 法结果对比。
- 子样本排名稳定性交叉验证。

### 1.5 图表建议
- 指标权重条形图、区域得分热力图/地图。

## 2. Q2：关键拥堵影响因素识别

### 2.1 数据适配性
- 目标：区域日均拥堵指数（连续）
- 特征：21 个候选因素，样本量 120，VIF<5，无强共线性。

### 2.2 主模型：XGBoost 回归 + SHAP 解释
- **模型族**：机器学习（梯度提升树）
- **输入**：21 个特征；输出：拥堵指数
- **输出**：特征重要性、SHAP 值
- **核心公式**：评估指标 RMSE、MAE（F007），暂不登记学习算法细节。
- **可解释性**：SHAP 提供一致性局部/全局解释。
- **复现**：固定种子 42，记录超参数。
- **失败后备选**：若过拟合，降级为 Lasso 回归或随机森林（Q2_FB1）。

### 2.3 备选模型
- Lasso 回归（线性基准）
- 随机森林（树集成）

### 2.4 验证方案
- 8:2 划分 + 5 折交叉验证
- 残差诊断：正态性、异方差性
- Bootstrap 特征重要性置信区间
- 空间独立性检查

### 2.5 图表建议
- 特征重要性对比条形图、SHAP summary dot plot、实际 vs 预测散点图。

## 3. Q3：交通拥堵缓解资源优化配置

### 3.1 数据适配性
- 基于 Q1、Q2 结果确定优化区域和措施。
- 效果系数由 Q2 模型弹性给出或情景模拟估算。

### 3.2 主模型：整数线性规划（ILP）
- **模型族**：优化决策
- **决策变量**：`x_k`，措施 k 的实施数量（整数）
- **目标**：最大化总效果 `sum(e_k * x_k)`
- **约束**：预算约束 `sum(c_k * x_k) ≤ B` 及措施互斥等
- **核心公式**：目标函数（F005）、预算约束（F006）
- **复现**：使用 PuLP/ortools 求解器，给定输入得确定解。
- **失败后备选**：若不可行或规模大，改用遗传算法（GA）或多目标 NSGA-II（Q3_FB1）。

### 3.3 备选模型
- 遗传算法（非线性/大规模）
- 鲁棒优化（系数不确定时）

### 3.4 验证方案
- 与基线方案（均匀分配）对比
- 预算/效果系数变化的场景分析
- 约束敏感性分析

### 3.5 图表建议
- 预算分配流图、优化前后对比柱状图、情景分析雷达图。
