# 符号说明

## 通用
| 符号 | 含义 |
|------|------|
| `n` | 评价对象数量 |
| `m` | 指标数量 |
| `i`, `j` | 索引 |

## Q1：综合评价
| 符号 | 含义 |
|------|------|
| `X = (x_{ij})` | 原始数据矩阵 |
| `z_{ij}` | 标准化值 |
| `p_{ij}` | 比重 |
| `E_j` | 信息熵 |
| `w_j` | 指标权重 |
| `v_{ij}` | 加权标准化值 |
| `d_i^+`, `d_i^-` | 到正、负理想解距离 |
| `C_i` | 相对贴近度 |

## Q2：因素识别
| 符号 | 含义 |
|------|------|
| `y_i` | 实际拥堵指数 |
| `ŷ_i` | 预测拥堵指数 |
| `f_i` | 特征向量 |
| `φ_{ij}` | 特征 j 对样本 i 的 SHAP 值 |

## Q3：优化
| 符号 | 含义 |
|------|------|
| `x_k` | 决策变量，措施 k 实施数量 |
| `e_k` | 措施 k 单位效果 |
| `c_k` | 措施 k 单位成本 |
| `B` | 总预算 |
| `K` | 措施总数 |

注：LaTeX 公式细节见 `14_contracts/formula_contract.csv`。
