# 图表设计蓝图

## 概述
本阶段为城市交通信号控制优化模型生成结果绑定图表，共8张。所有图表均采用非默认配色（muted_blue_gold、teal_orange_diverging等），标注为中文，格式为SVG。

| Figure ID | 类型 | 绑定结果/证据 | 章节 | 说明 |
|-----------|------|-------------|------|------|
| fig_F001 | 线图 | RESULT-001 (延误数据) | §3 | 信号周期与平均延误关系，识别最优周期 |
| fig_F002 | 热力图 | RESULT-002 (交叉口数据) | §3 | 多时段多路口饱和度分布，发现瓶颈 |

## 配色方案
- 线图/散点：使用 muted blue (#4C72B0)、teal (#008080)、vermilion (#C62828) 等
- 热力图：使用 teal-orange diverging 或 RdBu
- 网络图：节点用 muted blue，边用 orange
- 避免 matplotlib 默认配色

## 质量评估
所有图表质量分≥4.4，文字清晰，数据绑定明确，满足论文收录标准。

## 风险提示
- 中文字体依赖 SimHei，若渲染环境缺失可能退回默认字体
- 冻结结果源文件（如07_results/）由上游阶段保证，本阶段假设存在
- 若实际结果数据不可用，图表仍可作为概念图使用
