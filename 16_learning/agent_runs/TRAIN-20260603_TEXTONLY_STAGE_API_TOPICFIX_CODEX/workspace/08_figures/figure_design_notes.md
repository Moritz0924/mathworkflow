# 图表设计说明

## 设计原则
- 所有图表必须绑定冻结结果或论断，服务于建模论证。
- 图表类型根据模型族选择：统计评价、预测回归、机器学习。
- 配色优先使用非默认调色板：muted_blue_gold, teal_orange_diverging, deep_blue_vermilion, nature_green_gold。
- 所有文字标注、标题、轴标签必须为中文。
- 导出格式为SVG矢量图，确保可缩放。

## 去冲突检查
- 无重复同质图：每个结果只绑定一张核心图。
- 类型分布合理：涵盖分布、相关性、预测、诊断、重要性、比较等多种类型。
- 图、表、公式职责清晰：图展示趋势和比较，表展示精确数值，公式展示模型结构。
