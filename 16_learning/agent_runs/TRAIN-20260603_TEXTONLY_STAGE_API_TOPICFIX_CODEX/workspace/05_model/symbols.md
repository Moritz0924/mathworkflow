# Symbols and Notation (Draft)

This document defines symbols used across all sub-questions. Symbols must be finalised before formula contract entries are approved.

## General

| Symbol | Description | Unit / Value Range |
|---|---|---|
| \(N\) | Number of routes | Integer > 0 |
| \(M\) | Number of criteria in evaluation | Integer > 1 |
| \(x_{ij}\) | Value of route \(i\) on criterion \(j\) | Continuous |
| \(w_j\) | Weight of criterion \(j\) | [0,1], sum = 1 |
| \(C_i\) | Composite score for route \(i\) | [0,1] |
| \(K\) | Number of demand nodes (stops) | Integer |
| \(d_{uv}\) | Demand from \(u\) to \(v\) | Passengers/day |
| \(t_{uv}\) | Travel time from \(u\) to \(v\) | Minutes |
| \(h_l\) | Headway of route \(l\) | Minutes |
| \(f_l\) | Frequency of route \(l\) | Vehicles/hour |
| \(Z_1\) | Objective 1: Total operating cost | Monetary units/day |
| \(Z_2\) | Objective 2: Average passenger waiting time | Minutes |
| \(T\) | Time periods in forecast | Days |
| \(y_t\) | Actual ridership at time \(t\) | Passengers/day |
| \(\hat{y}_t\) | Predicted ridership | Passengers/day |
