# 缺失信息与待确认事项

## 1. 题面完整性
- [ ] 真实题面文件无法读取或未提供（训练沙箱限制）。
- [ ] 题目编号、比赛名称、数据来源、提交方式未知。
- [ ] 问题原文的歧义处理准则未得到人工确认。

## 2. 变量与单位
- [ ] 所有变量的物理/业务含义未确认。
- [ ] 单位体系（如国际单位制、货币单位、时间分辨率）未确认。
- [ ] 变量间因果关系或相关性未从题面中提取。

## 3. 约束与目标函数
- [ ] 显式数学约束未列出。
- [ ] 目标函数是否为单目标、多目标转化为单目标不得而知。

## 4. 数据质量与附件
- [ ] 训练集与测试集划分方式、是否允许使用外部数据未知。
- [ ] 数据缺失值处理规则、异常值定义由官方给出与否未确认。

## 5. 交付物与评分
- [ ] 论文格式模板（如 LaTeX 类文件）未提供或未约束。
- [ ] 提交文件列表（论文、代码、结果表格等）未明确。
- [ ] 评分细则未给出。

## 6. 风险清单
| 风险 | 严重程度 | 说明 |
|------|----------|------|
| 题面不可读 | 阻塞 | 无法启动 EDA 和任务拆解 |
| 附件字段不明 | 中 | 即使数据可读，字段含义未知也无法分析 |
| 单位口径冲突 | 中 | 不同数据源单位可能不一致，需确认 |
| 提交格式未知 | 高 | 最终交付格式可能不符合要求 |
| 评分暗示缺失 | 低 | 难以针对性优化论文权重 |

## 7. 需要人工确认的问题
1. 是否提供完整的题面 PDF 或文本？
2. 变量单位与数据说明文档是否完整？
3. 对题面中可能的歧义解释，应采纳哪一种？
4. 是否允许额外数据源？
5. 最终提交需要哪些文件？

> 本报告基于训练沙箱生成，所有条目均为占位信息，真实缺失情况需在实际场景下评审。
