# 图表类型库 v2.4

## 高优先级图类

| 模型/任务 | 优先图类 | 用途 |
|---|---|---|
| 综合框架 | 总技术路线图、模型链路图、变量作用机制图 | 让评委快速看到论文骨架 |
| 评价排序 | 指标体系图、权重热力图、综合得分矩阵、平行坐标、排序坡度图 | 解释指标、权重、排序变化 |
| 优化决策 | 目标约束结构图、可行域图、路径规划图、Pareto 前沿、甘特图、敏感性热力图 | 解释目标、约束、方案质量 |
| 预测回归 | 变量关系热力图、趋势分解图、预测区间图、残差诊断组图、误差分布、特征重要性图 | 解释关系、预测可信度、误差来源 |
| 机理仿真 | 机制示意图、状态转移图、仿真轨迹图、相图、参数敏感性曲面、情景对比图 | 解释模型机理和动态结果 |
| 机器学习 | 模型结构图、混淆矩阵、ROC/PR、特征重要性、消融实验图、误差热力图 | 解释性能、泛化和可解释性 |
| 空间网络 | 分级设色地图、点密度地图、流向图、网络社区图、桑基图、路径图 | 解释空间分布、流量、连接关系 |

## 降级使用图类

柱状图、折线图、普通散点图只在以下情况使用：

1. 数据维度很低，且高级图会制造噪声。
2. 用于附录或中间诊断。
3. 经过设计增强，如排序条形图、坡度图、带置信区间折线、带注释基准线散点。

## 禁止项

- 默认蓝色折线图。
- 无排序的普通柱状图。
- 无单位、无注释、无结论的热力图。
- 颜色超过信息维度。
- 中文乱码或负号乱码。
