# Skill Output Templates

Use `generic_input_manifest.yaml` and `generic_promotion_notes.md` for any skill
call. Use `by_category/<category>/` when the output category has extra binding
or validation rules.

Categories:

```text
literature_search
citation_verification
data_reproducibility
prior_reading
figure_design
section_writing
review_response
language_polish
post_final_presentation
```

Template rule:

```text
input_manifest.yaml describes what was read and whether the call was allowed.
promotion_notes.md describes whether archived output can be used downstream.
```
